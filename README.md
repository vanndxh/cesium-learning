# Cesium-Learning

## Introduction
利用Cesium对浙江省地图进行一系列操作，学习练手作品。

- 基础功能
  - 地图影像加载
  - 地图注记加载
  - 3d tiles加载
  - 地形加载
- 复杂功能
  - 视域分析
  - 通视分析
  - 日照分析
- 其他功能
  - 鼠标位置实时显示经纬度
  - 按键移动camera
